# BANOV – Server Hosting Rules & FAQ

Thank you for supporting Banov and being part of the community.
These rules exist to protect the project while giving server owners enough freedom to create their own experience.

## 1. Map Usage

You may host public or private servers using Banov.

## 2. Custom Locations & Building

**Allowed.**
Server owners may create their own locations, compounds, events, or gameplay areas on top of Banov.

## 3. Repacking / Reuploading Banov

**Not allowed.**
Repacking, reuploading, redistributing, mirroring, or publishing modified Banov map files without permission from the Banov Team is strictly prohibited.

## 4. Editing Banov Scripts / Map Systems

**Not allowed unless explicitly approved.**
Editing, replacing, removing, or modifying scripts, systems, objects, configurations, or features directly linked to Banov is not permitted.

## 5. Banov Branding & Main Menu

**Must remain intact.**
Original Banov branding elements (including Main Menu, presentation elements, credits, and identity components) may not be removed, hidden, or modified without prior approval.

## 6. VIP / Monetization

**Allowed.**
Server owners may sell VIP access, supporter packages, cosmetic benefits, and similar server features as long as they follow DayZ monetization policies.

## 7. Adding New Types / Expanding Content

**Allowed.**
You may add new items, vehicles, loot types, events, and compatible mods into the existing Banov setup.

## 8. Server Performance

Recommended player capacity:
**60–120 players**
(Actual performance depends on mods, hardware, and server configuration.)

## 9. Bug Reports

If you discover issues related to Banov, please report them to the Banov Team instead of distributing modified fixes.

## 10. Respect for the Project

Do not present modified versions as official Banov releases.

# Frequently Asked Questions (FAQ)

### Q: Can I add my own locations on the map?

**A:** Yes.

### Q: Can I repack Banov?

**A:** No. Repacking and redistribution are not allowed.

### Q: Can I edit scripts, objects, etc. that are linked to the map?

**A:** No, unless approved by the Banov Team.

### Q: Can I sell VIP slots on the Banov map server?

**A:** Yes, as long as it follows DayZ monetization rules.

### Q: What is the suggested player count for Banov?

**A:** 60–120 players.

### Q: Can I add new types to the existing vanilla Banov setup?

**A:** Yes.

### Q: Can I remove the Banov Main Menu?

**A:** No. Original Banov branding and presentation should remain intact.
